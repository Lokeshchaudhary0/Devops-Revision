# Azure DevOps End-to-End Deployment

## Project Overview
This project demonstrates complete CI/CD automation using Azure DevOps, Docker, AKS, Helm and ArgoCD.

## Architecture

Developer
↓
GitHub
↓
Azure DevOps Pipeline
↓
Docker Build
↓
ACR
↓
AKS
↓
Helm
↓
ArgoCD
↓
Prometheus/Grafana

- Azure DevOps

- Docker
- Kubernetes
- Helm

- Prometheus
- Grafana

## Features
- CI/CD Automation
- GitOps Deployment
